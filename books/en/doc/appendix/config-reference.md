# Config Reference (User Scope)

## TL;DR
- This page lists only the settings most frequently used by developers.

## Gateway
- `EVM_CANISTER_ID` (required)
- `RPC_GATEWAY_IC_HOST`
- `RPC_GATEWAY_PORT`

## Indexer
- `EVM_CANISTER_ID` (required)
- `INDEXER_DATABASE_URL` (required)
- `INDEXER_IC_HOST`
- `INDEXER_CHAIN_ID` (default: `4801360`)
- `INDEXER_MAX_BYTES`
